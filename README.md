# Brant Beach Plumbing Website

Static website for Brant Beach Plumbing, ready to publish with GitHub Pages.

## Files

- `index.html` - site content and structure
- `styles.css` - responsive styling
- `script.js` - mobile menu behavior

## Publish On GitHub Pages

1. Create a new GitHub repository.
2. Upload these files to the root of the repository.
3. In GitHub, open `Settings` > `Pages`.
4. Set the source to the main branch and root folder.
5. Save and wait for GitHub to publish the site.

The quote form uses the visitor's email app to send a message to `mylbiplumber@gmail.com`.
